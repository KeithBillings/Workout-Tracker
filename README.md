## Installation: 

To use this app on your local computer 

* Clone the repo

* Run the command ``` npm i ``` to install dependancies 

* Seed the project with the command ``` npm run seed ``` 

* To run the server ``` npm start ``` 
    
## Usage: 
    
To keep track of a user's workouts
